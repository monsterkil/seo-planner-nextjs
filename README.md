# SEO Planner (Next.js)

Aplikacja do planowania kampanii PBN — wizualizacja przepływu linków, fraz, anchorów i timeline. Wersja Next.js 15 z App Router.

## Uruchomienie

```bash
npm install
npm run dev
```

Otwórz [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Vercel

Projekt jest gotowy do wdrożenia na [Vercel](https://vercel.com) — po podłączeniu repozytorium GitHub deployment odbywa się automatycznie.
